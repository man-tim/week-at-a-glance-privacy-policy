Privacy Policy

Prokeep Week at a Glance — Chrome Extension
Last updated: April 21, 2026

1. Overview

The Prokeep Week at a Glance ("the Extension") is a Chrome extension that provides a live, simplified overview of your upcoming customer calls directly from your Google Calendar. This privacy policy describes the Extension's data practices.

Summary: This Extension requests read-only access to your Google Calendar to display your schedule. It does not permanently store, share, or transmit your calendar data to any external third-party servers. It requests the "identity" permission strictly for secure authentication.

2. Data Collection

The Extension fetches calendar event data (specifically event times and titles) from your Google Calendar API directly to your device to display your schedule. Specifically:

  • No personally identifiable information is collected beyond the Google OAuth token required for secure authentication.
  • No browsing history, cookies, or website content is accessed or collected.
  • No financial data or health information is collected.
  • No analytics, telemetry, or usage tracking of any kind is implemented.
  • Calendar data fetched by the Extension is processed in browser memory during the active popup session and is automatically discarded when the popup closes. It is never written to disk or transmitted anywhere.

3. Data Storage

The Extension does not permanently store any user data. Google Calendar data is requested on-demand when you open the Extension and is only held in temporary memory while the popup remains open. The Extension does not utilize local storage, session storage, or external databases to save your schedule.

4. Data Transmission and Sharing

The Extension makes direct network requests to Google's Calendar API (googleapis.com) to retrieve your schedule. Aside from this direct connection to Google, the Extension does not transmit any data to any other server, API, or third party.

The Extension does not:

  • Send data to any external servers, including Prokeep's own servers.
  • Use any analytics, tracking, or advertising services.
  • Share, sell, rent, or transfer any data to any third party for any purpose.
  • Use remote code or load any external scripts.

5. Permissions

The Extension requests the "identity" permission. This is used exclusively to securely authenticate you with your Google Workspace account using OAuth2 and request read-only access to your primary Google Calendar. It does not access tabs, browsing history, cookies, website content, or any other browser data.

6. Data Retention

Since the Extension does not permanently store any user data, there is no user data to retain. In-memory calendar data from the active session is discarded immediately when the popup closes. Uninstalling the Extension removes it entirely from the device.

7. User Controls

The Extension requires you to authenticate with your Google Workspace account to function. Users can revoke the Extension's access to their Google account at any time through their Google Account security settings. Users can also uninstall the Extension at any time through Chrome's extension management page (chrome://extensions), which removes all associated files.

8. Children's Privacy

The Extension is a workplace productivity tool and is not directed at children under the age of 13. It does not knowingly collect any information from any user, including children.

9. Changes to This Policy

Updates to this privacy policy will be reflected in a new version of the Extension published to the Chrome Web Store, with the "Last updated" date revised accordingly.
